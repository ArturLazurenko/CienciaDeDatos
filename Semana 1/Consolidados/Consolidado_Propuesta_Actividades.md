## Actividad 1.1

### ¿Qué es la ciencia de datos?
- La ciencia de datos es un campo interdisciplinario que combina estadística, matemáticas, inteligencia artificial e ingeniería para extraer conocimientos valiosos de datos estructurados y no estructurados

- Componentes principales: Datos, Estadistica, Programacion, Machine Learning y Visualizacion

### Datos estructurados
- Tienen un formato organizado (tablas, filas, columnas)
- Son fáciles de almacenar y analizar
- **Ejemplo:** una base de datos de clientes en Excel

### Datos no estructurados
- No tienen un formato definido
- Son más difíciles de procesar
- **Ejemplo:** imágenes, videos, audios, publicaciones en redes sociales



### Las 5 V del Big Data

-  Volumen
    - Gran cantidad de datos  

-  Velocidad
    - Rapidez con la que se generan y procesan datos  

-  Variedad
    - Diferentes tipos de datos  

-  Veracidad
    - Calidad y confiabilidad de los datos  

-  Valor
    - Utilidad de los datos para tomar decisiones

- **Spotify**

    - Tipo de datos que recopilan:

¿Qué es Kaggle y cómo puedes usarlo?
Kaggle es una plataforma en línea enfocada en la ciencia de datos, el aprendizaje automático y el análisis de datos.

¿Para qué sirve?
- Encontrar datasets públicos  
- Practicar análisis de datos y machine learning  

Dataset elegido: Video Game Sales (1980–2024)

Tipo de datos:
- Datos estructurados (tablas)  
- Datos numéricos (ventas) y categóricos (género, plataforma)  

¿Qué información contiene?
- Nombre del videojuego  
- Plataforma (PS, Xbox, PC, etc.)  
- Género  
- Desarrolladora  
- Empresa (publisher)  
- Ventas Totales  
- Año de lanzamiento  

¿Qué preguntas se pueden responder?
- ¿Qué juegos son los más vendidos?  
- ¿Qué plataformas generan más ventas?  
- ¿Qué géneros son más populares?
